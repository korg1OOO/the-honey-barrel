The Honey Barrel - Privacy Policy

Welcome to The Honey Barrel, a Chrome extension designed to help whisky and wine enthusiasts find better deals by comparing prices on retail websites with the BAXUS marketplace. This Privacy Policy explains how we handle data, ensuring transparency and prioritizing your privacy.

1. Data We Access

The Honey Barrel extension operates by scraping publicly available bottle information (e.g., name, price, size, ABV) from supported retail websites when you visit a product page. This data is processed locally in your browser to compare with listings on the BAXUS marketplace.

We do not access, collect, or process any personal data such as your name, email, browsing history, or any other personally identifiable information.

2. Third-Party Services

The extension makes the following network requests:

  • BAXUS API: We query the BAXUS marketplace API (https://services.baxus.co/api/search/listings) to retrieve listings for price comparison. Only bottle-related data is accessed, and no personal data is shared with BAXUS.
  • ExchangeRate-API: We fetch exchange rates from ExchangeRate-API (https://api.exchangerate-api.com) to convert prices to USD for accurate comparisons. No personal data is sent in these requests.

We recommend reviewing the privacy policies of these third-party services for more information on their data practices.

3. Data Storage

The extension uses Chrome’s storage.local API to cache BAXUS marketplace listings for performance optimization. This cache is stored locally on your device and contains only bottle-related data (e.g., names, prices, sizes). The cache expires after 24 hours and is automatically refreshed.

No personal data is stored, and the cached data is not transmitted to any external servers.

4. Data Sharing

We do not share any data with third parties, except for the API requests mentioned above, which do not include personal information. All processing occurs within your browser.

5. Permissions

The extension requests the following permissions:

  • activeTab: To access the current webpage and scrape bottle information.
  • storage: To cache BAXUS listings locally for faster performance.
  • scripting: To inject scripts into supported retail websites for scraping.
  • host_permissions: To access specific retail websites and the BAXUS/ExchangeRate APIs.

These permissions are the minimum required to provide the extension’s functionality and are used solely for the purposes described in this policy.

6. User Rights

Since The Honey Barrel does not collect personal data, there is no user data to manage. You can uninstall the extension at any time via Chrome’s extension settings, which will remove all locally cached data.

7. Changes to This Policy

We may update this Privacy Policy from time to time to reflect changes in the extension’s functionality or legal requirements. Any updates will be reflected on this page with a revised "Last Updated" date.
